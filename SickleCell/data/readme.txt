----------------- SCD model -----------------

Hb_Ribeil.csv and RBC_Ribeil.csv are data published from Ribeil et al., 2017. 
https://www.nejm.org/doi/full/10.1056/nejmoa1609677


----------------- beta thalassemia model -----------------

HGB20*_HbAT87Q_*_Thompson.csv are digitized from Thompton et al., 2018.

----------------- pyruvate kinase deficiency (PKD) model -----------------

PKD_PKac_RET.csv is the RET count (*10^9/L, y axis) and PK activity (x axis); data from Zanella et al., 2005 
https://onlinelibrary.wiley.com/doi/10.1111/j.1365-2141.2005.05527.x
